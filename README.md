# WiFi QR Scanner

A minimal Windows app that opens your webcam, scans a WiFi QR code, and connects your PC to the network automatically.

## Install

Open **PowerShell as Administrator** and run:

```powershell
irm https://raw.githubusercontent.com/mkster/wifi-qr-scanner/main/install.ps1 | iex
```

That's it. No manual downloads, no .NET installation needed. The app will appear in your Start Menu.

## Requirements

- Windows 10 or 11
- A webcam

## How to use

1. Launch **WiFi QR Scanner** from the Start Menu
2. Accept the UAC prompt (needed to connect to WiFi via `netsh`)
3. Point your camera at a WiFi QR code
4. The app connects automatically — password shown in the status bar

### Supported QR formats

Generated by most router apps, phone hotspot screens, or any QR generator that supports the WiFi format:

```
WIFI:T:WPA2;S:NetworkName;P:Password;;
WIFI:T:WPA;S:NetworkName;P:Password;H:true;;
WIFI:T:WEP;S:NetworkName;P:Password;;
WIFI:T:nopass;S:OpenNetwork;;
```

## Build from source

Requires [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0).

```powershell
git clone https://github.com/mkster/wifi-qr-scanner
cd wifi-qr-scanner
dotnet run
```

To publish a release build:

```powershell
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=false -o ./publish
```

## Release a new version

Push a version tag and GitHub Actions builds and publishes the release automatically:

```powershell
git tag v1.0.0
git push origin v1.0.0
```

## Tech stack

- C# / WPF (.NET 8)
- [OpenCvSharp4](https://github.com/shimat/opencvsharp) — camera capture
- [ZXing.Net](https://github.com/micjahn/ZXing.Net) — QR decoding
- `netsh wlan` — WiFi connection
